# Test Automation Project (Java)

Командный проект по автоматизированному тестированию на Java.

---

## 🎯 О проекте

В рамках проекта разрабатывались автоматизированные тесты для веб-приложения.

Работа велась в команде с использованием Git и совместной разработки.

---

## 🛠 Стек

- Java  
- Selenium WebDriver  
- JUnit / TestNG  
- Maven  
- Git  

---

## 🧪 Что реализовано

- UI-автотесты  
- проверка пользовательских сценариев  
- работа с элементами страницы  
- базовая структура тестового проекта  

---

## 👥 Моя роль

- разработка автотестов  
- работа с локаторами  
- участие в написании тестовых сценариев  
- работа с Git (commit, push, merge)  

---

## 📁 Структура проекта

- `src` — исходный код тестов  
- `.github/workflows` — CI / workflow  
- `pom.xml` — зависимости Maven  

---

## ▶️ Запуск

1. Установить Java  
2. Установить Maven  
3. Скачать ChromeDriver  
4. Выполнить:

```bash
mvn test
