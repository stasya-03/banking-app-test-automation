# Bank Account Testing Project

Командный проект по тестированию приложения для операций с банковскими счетами.

---

## 🎯 О проекте

В рамках проекта выполнялось тестирование логики работы банковского приложения, включающего операции со сберегательными и кредитными счетами.

Проект выполнялся в команде с использованием Git и совместной разработки.

---

## 📌 Что входило в задачи

- тестирование части функциональности приложения  
- поиск дефектов  
- составление баг-репортов  
- исправление найденных дефектов  
- закрытие баг-репортов коллег  
- работа с GitHub в командном формате  

---

## 🛠 Стек

- Java  
- JUnit  
- Maven  
- Git / GitHub  

---

## 🧪 Что тестировалось

Приложение для операций с банковскими счетами:

- `SavingAccount`
- `CreditAccount`
- операции `add`
- операции `pay`
- расчёт `yearChange`
- перевод между счетами (`transfer`)  

---

## 👥 Моя роль

- тестирование логики приложения  
- анализ поведения методов  
- поиск и оформление дефектов  
- исправление багов в коде  
- работа с ветками, commit, push, pull request и merge
